Redesigning an online tennis shop in Hong Kong

Part 1

Picking a website

As a varsity tennis player from Hong Kong, I’ve frequently purchased tennis apparel/shoes from Tennis Star HK and since “Jeff Tennis” (the owner of the shop who is not a CS major) developed his own website, I’ve noticed that there were many issues with the website that could be fixed to improve user experience.

Link to website

URL: https://www.tennisstarhk.com/adidas-men-s-shoes

Finding problems

  • Branding and memorability issue as shop name and location are not on front header
  • When users click on a product to try and purchase, they are only shown an enlarged photo of the product (nothing to add to cart or steps on how to purchase)
  • There is no shopping cart to add items users want to purchase
  • The purchasing process is not clearly labelled
  • There is no sort function for products (sorting based on price or popularity)
  • Shoes of all brands should be displayed as it is inconvenient for users to have to go through different brand pages just to find the shoes they want to buy
  • There is no rating system for products, so users are not able to gauge product quality
  • The only language available is English which makes the website hard to use, especially considering that most Hong Kong people can only read simplified/traditional Chinese

Picking accessibility

  • There were 22 contrast errors all pointing to the text and its background which make it difficult for users (especially color blind users) to read. The color themes and schemes of this website need to be changed.
  • The website is only available in English. Considering that a large percentage of the Hong Kong population have Cantonese as their first language, it would be wise to add an optionality for simplified/traditional Chinese to increase accessibility
  • There are a large number of headers of equal size which make elements of the website and their importance hard to comprehend, especially for blind users who rely on hearing software
